# Arrays

Arrays are one of the most fundamental data structures and form the basis for many interview questions.

---

## Common Patterns

- Prefix Sum
- Difference Array
- Two Pointer
- Sliding Window
- Kadane's Algorithm
- Sorting
- Binary Search on Arrays

---

## Time Complexities

| Operation | Complexity |
|-----------|------------|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |
